$background default
$background Цвет
$background ИмяФайла 		Директива управления подложкой книги. Есть три варианта его использования:

1) Аргумент default - устанавливает подложку, которая у читателя книги выбрана подложкой по умолчанию для Города.

2) Аргумент Цвет - задает сплошную подложку указанного цвета.

3) Аргумент ИмяФайла - устанавливает фоновый рисунок в качестве подложки книги (этот вариант заявлен, но пока не поддерживается Городом).
(Детальнее см. Подложка)
$define Имя Значение Директива замены. Начиная со следующей строки исходного текста книги, все вхождения %Имя% будут заменены на Значение.
(Детальнее см. Замены)
$title Название Директива установки названия книги. Это название будет видно читателям на странице загрузки книги.
(Детальнее см. Название книги)
$vote Имя Определение Директива определения голосования. Голосований в одной книге может быть несколько. Параметр Имя задает имя голосования, которое будет использовано в тэгах голосования <vote:Имя> или <votep:Имя>.
Параметр Определение формируется из вопроса и всех вариантов ответа, разделенных точкой с запятой. Эта информация используется Городом на странице показа результатов данного голосования.
(Детальнее см. Голосования)
$voteto ДатаВремя Директива устанаваливает срок окончания голосования.
Аргумент ДатаВремя задается в формате ГГГГММДДЧЧММСС (ГГГГ - год, ММ - месяц, ДД - день, ЧЧ - часы, ММ - минуты, СС - секунды), время московское.
Директива действует на все голосования, объявленные директивой $vote ниже по тексту, и не действует на те, которые объявлены выше.
(Детальнее см. Голосования)

<if:Значение>
<elseif:Значение>
<else>
<endif> 		Группа тэгов <if> - <endif> представляет из себя структуру альтернативного выбора одного из нескольких вариантов исходного текста книги, в зависимости от текущего значения счетчика викторины. Аргумент Значение тэгов <if> и <elseif> определяет значение счетчика, при совпадении с которым будет показан следующий за данным тэгом текст. Значение можно задавать как одним чилом, там и парой чисел (через дефис), определяя сразу целый диапазон. Тэг <else> определяет вариант, который будет выбран, если ни одно из значений тэгов <if> и <elseif> не совпало со значением счетчика. Тэг <endif> закрывает группу.
В общем виде структура выглядит так:

   <if:Значение1>
      Вариант текста для Значение1.
   <elseif:Значение2>
      Вариант текста для Значение2.
   <elseif:Значение3>
      Вариант текста для Значение3.
   ...
   <elseif:ЗначениеN>
      Вариант текста для ЗначениеN.
   <else>
      Вариант текста, если ни одно из значений не совпало.
   <endif>

(Детальнее см. Счетчик викторины)
<ifmask:Значение>
<elseifmask:Значение>
<else>
<endif> 		Группа тэгов <ifmask> - <endif> представляет из себя структуру альтернативного выбора одного из нескольких вариантов исходного текста книги, в зависимости от текущего значения маски викторины. Аргумент Значение тэгов <if> и <elseifmask> определяет битовую маску (число в 16-ричной записи), при логическом умножении (операция AND) которой на текущее значение маски викторины, если результат ненулевой - будет показан следующий за данным тэгом текст. Значение можно задавать как одним чилом, там и парой чисел (через дефис), определяя сразу целый диапазон. Тэг <else> определяет вариант, который будет выбран, если ни одно из значений тэгов <ifmask> и <elseifmask> не дало ненулевой результат при логическом умножении на текущее значение маски викторины. Тэг <endif> закрывает группу.
В общем виде структура выглядит так:

   <ifmask:Значение1>
      Вариант текста для Значение1.
   <elseifmask:Значение2>
      Вариант текста для Значение2.
   <elseifmask:Значение3>
      Вариант текста для Значение3.
   ...
   <elseifmask:ЗначениеN>
      Вариант текста для ЗначениеN.
   <else>
      Вариант текста, если ни одно из значений не совпало.
   <endif>

(Данная версия Редактора эту группу тэгов не обрабатывает).
(Детальнее см. Маска викторины)
<img:ИмяКартинки>
<img:Номер:Ширина> 		Тэги <img:ИмяКартинки> и <img:Номер:Ширина> размещают непосредственно в книге картинку. Выбор варианта тэга зависит от выбранного стиля книг (см. Стиль книг).

Для стиля книг "Обычная книга" используется тэг <img:ИмяКартинки>. Картинка задается именем файла, файл автоматически пакуется в архив перед отправкой книги в Город.

Тэг <img:Номер:Ширина> используется для стиля книг "Как с телефона". Аргумент Номер определяет номер вложения (которое будет автоматически прикреплено к письму при отправке книги в Город), аргумент Ширина указывает максимальную ширину картинки. Если ширина картинки не превышает указанное значение, то картинка будет размещена в книге без каких-либо изменений. Если же ширина превышает значение аргумента, сервер Города произведет масштабирование, ужимая картинку (пропорционально по горизонтали и по вертикали) до указанного значения ширины. Максимально допустимая Городом ширина картинки составляет 128.
(Детальнее см. Картинки)

<small>текст</small> Текст, помеженный между тэгами <small> и </small>, будет иметь уменьшенный размер шрифта (пример: обычный шрифт, уменьшенный шрифт).
(Детальнее см. Размер шрифта)
<store> Тэг размещает в книге кнопку, при нажатии которой данная книга будет сохранена в хранилище на телефоне читателя.
(Детальнее см. Другие команды)
<u>текст</u> Текст, помеженный между тэгами <u> и </u>, будет выведен с использованием подчеркивания (пример: обычный текст, подчеркнутый текст).
(Детальнее см. Стиль шрифта)
<vote:Имя Номер> Тэг размещает в книге кнопку, при нажатии которой осуществляется обмен с Городом, и Город фиксирует голос данного читателя за вариант номер Номер голосования Имя.
(Детальнее см. Голосования)
<votep:Имя Номер> Тэг <votep:Имя Номер> размещает в книге кнопку, при нажатии которой осуществляется обмен с Городом, и Город фиксирует голос данного читателя за вариант номер Номер голосования Имя.
В отличие от тэга <vote:Имя Номер>, принять участие в данном голосовании могут только жители, баланс "жизней" которых больше ноля. (Детальнее см. Голосования)
<writetop> Тэг размещает в книге кнопку, при нажатии которой результат прохождения викторины данным читателем будет записан на сервере Города, если он попадает в десятку лучших.
(Детальнее см. Команды викторины)
<z:Тип:Вариант Имена> Тэг <z:Тип:Вариант Имена> размещает в книге кнопку, при нажати которой Город выполняет команду знакомства.

Аргумент Тип определяет тип знакомства:
   bg - юноша с девушкой;
   gb - девушка с юношей;
   bb - юноша с юношей;
   gg - девушка с девушкой;
   aa - житель с жителем.
Необязательный аргумент Вариант - любая буквенно-цифровая последовательность, для указания преимущественного выбора внутри книги знакомств.

Аргумент Имена - это список разделенных пробелом имен вариантов текстового сообщения, определенных ранее директивами $define, обрамлять имена знаком % здесь не нужно (данная версия Редактора не обрабатывает этот тэг).
(Детальнее см. Знакомства)
