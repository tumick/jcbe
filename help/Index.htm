Редактор книг

Программа Редактор книг предназначена для создания книг специального формата, предусмотренного программой для мобильных телефонов JetCity ("Город"). Редактор имеет синтаксическую подсветку тэгов, окно предварительного просмотра книг и функцию автоматической отправки готовой книги в Город. Ниже перечислены основные разделы справочной системы:

  • Управление в программе

  • Работа с текстовым редактором

  • Book-нотация Города
